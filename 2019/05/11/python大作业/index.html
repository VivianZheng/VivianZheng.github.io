Linux运行Python程序实现汇率转换

发表于 2019-05-11 | 分类于 测试笔记

需求：美元价格需要转化成欧元单位。

程序要求：

1、接收一个 CSV 文件为输入值
2、对价格这一列进行货币转换
3、输出这个文件的内容

限制条件：

1、csv 文件解析器要自己实现，不能用库
2、输出文件必须正确
3、可以把数据从欧元改回美金

例 1：

head data.csv

例 2：

./currency_convert –help
usage:currency_convert<–field N–multiplier N][-i input][-o output]

field N 价格信息在 CSV 的第 N 列，需要进行转换

multiplier N 转换方法为把当前的数值乘以 N N 是汇率

i input 从 input 文件内读取 CSV 文件内容（或者从 stdin 读入）

o output 输出到 output 文件中（或者输出 stdout）

linux 操作指令，规定了一些使用到的命名，以及表明程序使用环境为 linux
